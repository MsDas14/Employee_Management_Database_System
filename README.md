# Employee_Management_Database_System (CRUD)
Full-stack Web Application using SpringBoot &amp; React JS (RestFull API).


### Used Technologies & Tools
* Front-end
  * React JS
  * Bootstrap
* Back-end
  * SpringBoot 2.x.x (Making REST API) [Java 8]
  * MySQL
